# Ikony pre Coffit

Sem vlož SVG, PNG alebo custom .dart ikony, ktoré chceš použiť v aplikácii.

Odporúčam používať SVG a balíček flutter_svg:
https://pub.dev/packages/flutter_svg

Príklad použitia SVG:
import 'package:flutter_svg/flutter_svg.dart';

SvgPicture.asset('lib/core/icons/meno_ikony.svg');
